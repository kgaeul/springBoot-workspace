static : 정적인 코드를 생성
정적(이미지,css,javaScript)를 저장하는 장소
서버에서 유동적으로 생성되는 것이 아니라 만들어진 그대로 클라이언트(고객, 사용자) 전달됨
IMG, CSS, JavaScript -> static 폴더에 저장 함

templetes : view에 표현되는 코드 작성
